Capabilities
Built for the real market
Every tool connects to live data. The bot actually learns. Broker integration is real, not simulated.
🧠
Real Machine Learning
200-tree RandomForest trained on 2 years of real OHLCV per ticker. 14 features: EMA ratios, RSI, MACD, ATR, Bollinger, candlestick patterns. 5-fold time-series cross-validation — no look-ahead bias. Predictions include confidence %, SL/TP levels, and feature importances.
📡
Live Market Data
All 11 dashboards pull real-time prices via Yahoo Finance. The Trading Bot fetches live intraday volume every 3 minutes to rank which instruments have the highest activity and trains hardest on those. Volume-ranked universe updates dynamically.
🔮
Ghost Candle Predictions
5 Monte Carlo scenarios rendered as translucent OHLC candles on the live chart. Each detects the dominant pattern (engulfing, doji, hammer), calculates SL/TP from ATR, shows R:R ratio. A BOT FOCUS box highlights the 5 candles the model is watching.
🏦
Interactive Brokers Integration
Paper trading on port 7497 — IBKR's simulator, zero real money. Switch to live (port 7496) after validation. Python bridge handles MKT/LMT/STP order routing, position monitoring, real-time P&L, auto-reconnect. TradingView webhook + Schwab OAuth 2.0 also wired in.
💾
Training Persists Forever
Training saves to Supabase every 5 minutes and on page close. IndexedDB provides a local backup. Patterns, edge scores, and session history survive browser clears, device switches, and deploys. Continues running when you switch tabs — background indicator shows live progress.
⚗️
Institutional-Grade Engines
Heston uses Milstein discretisation. Merton uses exact Poisson sampling. SABR absorbing boundary. Ledoit-Wolf covariance shrinkage. ADF, Johansen, Hurst for pairs. Nelson-Siegel + Svensson for yield curves. All run as serverless Python on Vercel — zero infrastructure to manage.
